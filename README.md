# SecurePass - Aplikasi Pengelola Kata Sandi Pribadi

Aplikasi SecurePass adalah program berbasis Command Line Interface (CLI) yang dirancang untuk menyimpan dan mengelola informasi login akun secara lokal dan aman. Proyek ini dikembangkan sebagai pemenuhan Tugas Besar mata kuliah Algoritma dan Pemrograman 2 di Fakultas Informatika, School of Computing, Telkom University.

## 🚀 Fitur Utama

- Manajemen Akun (CRUD): Menambah, melihat, mengubah, dan menghapus data akun layanan (nama layanan, email, kata sandi, dan tanggal pembaruan).
- Pencarian Data (Searching): Menemukan informasi akun berdasarkan nama layanan secara cepat menggunakan metode Sequential Search atau Binary Search.
- Pengurutan Data (Sorting): Mengurutkan daftar akun secara alfabetis berdasarkan nama layanan menggunakan algoritma Selection Sort atau Insertion Sort.
- Analisis & Statistik Kekuatan Sandi: Mengklasifikasikan kekuatan kata sandi (Lemah, Sedang, Kuat) berdasarkan variasi karakter dan panjangnya, serta menampilkan akumulasi statistiknya secara real-time.
